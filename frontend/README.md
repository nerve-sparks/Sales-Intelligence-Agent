# Agentic Sales Agent Frontend

React, TypeScript, Vite, and Tailwind CSS frontend for the Agentic Sales Agent project.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run lint`

## Structure

- `src/features` contains product flows and pages.
- `src/components` contains reusable UI and brand components.
- `src/lib` contains shared utilities.
- `src/assets` is reserved for exported Figma assets.

The Figma node currently needs editor access for the connected MCP account before the login screen can be made pixel-exact.
